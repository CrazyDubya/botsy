Tools & APIs Overview

Free Tier APIs

  • NewsAPI - 1000 requests/month
  • Alpha Vantage - 500 calls/day
  • OpenWeather - 1000 calls/day
  • GitHub API - 5000/hour with auth

Completely Free

  • RSS Feeds - Unlimited

Web Scraping

  • Beautiful Soup - HTML parsing

Getting Started

Installation

git clone https://github.com/CrazyDubya/botsy.git
cd botsy
pip install -r requirements.txt

Basic Usage

# Run a specific category
python main.py --category news

# Run all categories
python main.py --all

# Show available tools
python main.py --tools

Configuration

# Copy environment template
cp .env.example .env

# Add your API keys
NEWS_API_KEY=your_key_here
GITHUB_TOKEN=your_token_here
